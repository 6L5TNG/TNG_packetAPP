# 📡 TNG Packet APP
**아마추어 무선(Ham Radio)을 위한 차세대 멀티모드 디지털 교신 플랫폼**

이 프로젝트는 텍스트와 데이터를 오디오 신호(Audio Packet)로 변환하여 무전기로 송수신할 수 있게 해주는 **소프트웨어 모뎀(TNC)**입니다.

## 🚀 주요 기능 (개발 중)
- **멀티 모드 지원**: 나만의 통신 프로토콜을 플러그인처럼 추가 가능
- **크로스 플랫폼**: Windows, macOS, Linux (Raspberry Pi) 지원
- **자동 업데이트**: 런처를 통한 간편한 최신 버전 유지
- **안전한 백업 시스템**: 패치 전 자동 로컬 백업(Latest_Backup) 지원

## 📦 설치 및 실행
1. 파이썬(Python 3.8+) 설치
2. 의존성 설치: `pip install -r requirements.txt`
3. 실행: `python launcher.py`
